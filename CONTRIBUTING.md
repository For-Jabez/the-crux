This repository is the fixed origin for Aeturnix – the Perpetual Refinery System.

- All contributions must include full provenance.

- Every change must survive the V1 Gladius Dei checks described in v1\_parser\_spec.md.

The goal is stable, compounding improvement instead of recursive collapse.
